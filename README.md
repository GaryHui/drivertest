# 粤驾速记

面向广东科目一学员的移动端数字规则学习站。核心结构是：7 张数字总表、易错数字对比、错题急救与模拟练习。

直接打开 `index.html` 即可使用，无需安装依赖。学习进度保存在浏览器本地。

## 资料口径

- 《道路交通安全违法行为记分管理办法》（公安部令第 163 号）
- 《机动车驾驶证申领和使用规定》（公安部令第 172 号）
- 《中华人民共和国道路交通安全法》及实施条例

完整考试题库并非公开数据。本项目不声称收录“内部原题”，练习题依据公开法规整理，使用前应继续按最新法规复核。

## 本地题库同步

网站只读取 `data/question-bank.json`，运行时不依赖任何第三方接口。公安公开题库作为基础；极速数据或 ROLL 的合法导出结果可分别保存为 `data/imports/jisu.json`、`data/imports/roll.json`，然后执行：

```powershell
node scripts/sync-question-bank.js
```

同步过程会统一字段、按题干去重，并隔离包含旧“2分”记分表述等明确疑似旧规题。每题保存来源、导入日期、适用地区、更新时间和核验状态。自动筛查不能代替法规人工复核，因此未逐题确认的题目会明确标为 `pending`，不会伪装成“已核验”。
